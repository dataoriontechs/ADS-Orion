# **App Name**: ADS Orion

## Core Features:

- Secure User Authentication: Allow users to securely sign up and log in via Google, email/password, and enable password recovery for access to their personalized dashboard.
- Campaign Creation & Management: Users can upload various ad creatives (images, videos, text), select target platforms (Google, Meta, TikTok, etc.), and configure campaign parameters like budget, duration, and target audience demographics and interests.
- AI Ad Creative Assistant: A generative AI tool that automatically suggests compelling titles and drafts advertising texts, optimizing content for maximum impact and engagement.
- Real-time Campaign Dashboard: Display a modern dashboard providing users with real-time performance metrics for active and past campaigns, including impressions, clicks, conversions, budget expenditure, and key graphical data visualizations.
- User Financial System: Allow users to manage their account balance, add funds via simulated payment methods (e.g., PIX, credit card), and view their financial transaction history within the platform.
- Admin Panel & Overview: Provide administrators with a central panel to monitor global platform metrics, including total users, active campaigns, overall revenue, and approve or manage user accounts and campaigns.

## Style Guidelines:

- Color scheme: Dark mode, leveraging deep, technologically-inspired tones to evoke a premium and futuristic feel.
- Primary brand color: A vibrant electric violet (#8426FF), signifying innovation and high-tech elegance, used for key interactive elements and branding accents.
- Background color: A very dark, subtle purple-grey (#19161C), providing a sleek, deep canvas that allows vibrant elements to pop and reduces eye strain.
- Accent color: A softer, bright blue (#7A89FF), providing a complementary contrast for secondary highlights, alerts, and progressive actions.
- Headline font: 'Space Grotesk' (sans-serif), chosen for its modern, slightly futuristic, and bold appearance, perfect for tech-focused titles and major dashboard elements.
- Body font: 'Inter' (sans-serif), selected for its excellent legibility across various screen sizes and its clean, professional aesthetic, ensuring clarity in data tables and descriptions.
- Icons: Clean, minimalist, and geometric line-based icons that maintain a cohesive, high-tech, and modern appearance, enhancing usability without clutter.
- Layout structure: Emphasizes a modern, responsive design using 'cards' for data segmentation, generous spacing, and a visually clean grid system to manage complex information efficiently in a dark theme.
- Animations: Subtle, smooth transitions for user interactions, data updates, and loading states, alongside engaging animated charts and graphs to enhance the sense of real-time responsiveness and modernity.
